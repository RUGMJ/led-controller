# Led Controller
> A tool to control my Led Strip

## Arguments

- Set Effect (-s --set-effect)
    Sets the [effect](#effects)
- Kill (-k --kill)
    Kills the daemon
- Web (-w --web)
    Sets the status of [led.rugmj.dev](https://led.rugmj.dev)
- Test (-t --test)
    Tests the led strip Red -> Green -> Blue -> White -> Repeat
